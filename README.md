# Pi Rack HUD

A lightweight, headless system telemetry HUD for Raspberry Pis in a mini rack, using a 0.96" ST7735S IPS display.

## Features

- CPU and RAM usage with slim progress bars
- CPU temperature
- Network usage (download / upload)
- Nerd Font icons with crisp 2Ã— supersampling
- Auto-start on boot via systemd
- Designed for low resource usage so it doesn't interfere with services like Wazuh or Home Assistant
